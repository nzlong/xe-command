# xe-command 操作函数: 支持复制到剪贴板、打印..

## 通过NPM安装最新版本

``` shell
npm install xe-command --save
```

### 按需引入
``` shell
import { copy } from 'xe-command'

copy('复制该内容到剪贴板')
```

### 引入所有
``` shell
import * as XECommand from 'xe-command'

XECommand.copy('复制该内容到剪贴板')
```
